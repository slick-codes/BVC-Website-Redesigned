# Setup

## Step 1
> install visual studio codes

## Step 2
> install the Sass compiler extension 

## Step 3
> Open the settings.json and insert the following code
 ```json 
    {
    "liveSassCompile.settings.formats": [
        {
            "format": "expanded",
            "extensionName": ".css",
            "savePath": "/assets/styles/css"
        }
    ]
}
```
